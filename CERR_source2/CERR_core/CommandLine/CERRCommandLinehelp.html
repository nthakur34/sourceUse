CERR Command Line Help

Following are the commands that can be run from the command line on CERR. User can type these commands in the command line or click on the hyper line to execute them.

SLICE POSITIONING

  • To go to a slice number: go to [ slicenum ]

  • To go to the slice with the maximum dose in total plan: go to max

  • To go to a z value (or nearest slice): go to z [ z value ]

STRUCTURES

  • To show points in a structure: list structure [structnum] points [slicenum]

  • To create an ROI: create structure;

  • To delete an anatomical structure: del structure [num]

  • To find volume for an anatomical structure, type vol structure [num]

  • To find union/intersection volumes for anatomical structures, type getunionintersectionvol [structIndicesV]

  • To translate structure and store it as new structure translate

  • Rename structure using GUI rename

  • To get center of mass for a structure structcom [num]

  • To associate structure/Structures of one scan to another assocstr [structure num] to [scan num] Where structure num can be an a string of structure numbers with space between each number

DOSE

  • To create a dose distance plot, type ddp [structNum] [maximum distance of plot] [resolution in cm] [''max'' or ''min'' dose] ''

  • To get the dose and position of a point, type dose , then select the position on the axes with left mouse click.To exit use mouse right click.

  • To export DVH type exp [opt] [structNum] dose [doseNum] where [structNum] stands for the structure number as given under Structures menu '[opt]has to be a string suggesting you want absolute(abs) or Normalized(nor) DVH, [doseNum] stands for dose number given under Dose menu example to export absolute DVH for structure number 1 and dose number 1 this is what you would type -> "exp abs 1 dose 1" (no quotes)'

  • To delete a dose: del dose [num]

  • To convert an iso-dose level to structure: doseToStruct [doseNum] [doseLevel] [scanNum] where [doseNum] stands for the dose number, [doseLevel]stands for dose level, [scanNum] stands for scan number to associate structure with.

  • To crop a dose around a structure: cropDose [doseNum] [structureNum] [margin] where [doseNum] stands for the dose number, [structureNum]stands for structure number, [margin] stands for margin (cm) around the structure.

SCAN

  • To get the scan intensity and position of a point, type scan , then select the position on the axes with left mouse click.To exit use mouse right click.

  • To export IVH type exportIVH [opt] [structNum] scan [scanNum] where [structNum] stands for the structure number as given under Structures menu '[opt]has to be a string suggesting you want absolute(abs) or Normalized(nor) DVH, [scanNum] stands for scan number given under Scan menu example to export absolute DVH for structure number 1 and scan number 1 this is what you would type -> "exportIVH abs 1 scan 1" (no quotes)'

  • To crop a scan around a structure: cropScan [scanNum] [structureNum] [margin] where [scanNum] stands for the scan number, [structureNum]stands for structure number, [margin] stands for margin (cm) around the structure.

ROI

  • To show the points included in an ROI: mask [num]

  • To paint low and high dose regions within an ROI: mask [structNum] [doseNum] [prescriptionDose]

  • To Toggle the points on the surface of an ROI (on the contour): dshpoints [num]

OTHERS

  • To reset and use a new option value: set [optionName] i.e. optS.name in CERROptions.m. Example optS.loadCT = 'yes'

  • To toggle on/off print friendly mode type ''printmode''.

  • To raise the 3-D menu type 3d

  • To show a digital film, type: ''show film [num]'

  • To launch RPC film GUI rpcfilm

LIST

List command displays in the command window the fields associated with the list command example header, comment, slice etc

  • list header
  • list comment
  • list slice [ num ]
  • list structure [ num ]
  • list beam geome [ num ]
  • list dose [ num ]
  • list dvh [ num ]
  • list options
