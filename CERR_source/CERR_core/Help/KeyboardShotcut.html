CERR Keyboard ShortCuts

Following are the Keyboard Shortcuts for CERR.

  1. Up Arrow for Previous slice (Updates Navigation Montage)

  2. Down Arrow for Next Slice (Updates Navigation Montage)

  3. left Arrow for Navigation Montage left

  4. Right Arrow for Navigation Montage right

  5. B Toggles bookmark on current Slice in Navigation Montage

  6. Tilda (`) (key, next to the 1) Always calls LabBook to take CERR Snapshot.

  7. 'z' key to toggles zoom

  8. 'e' key contour 'editMode' mode

  9. 'd' key contour 'drawMode' mode

  10. 't' key contour 'threshMode' mode

  11. 'r' key contour 'reassignMode' mode
